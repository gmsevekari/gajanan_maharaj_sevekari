## Setup

* Download the latest Android Studio https://developer.android.com/studio
* Install and setup the project folder
* Use either Terminal or Android Studio to clone and open the project
  * Terminal
    * Open terminal `open -a "Terminal.app"`
    * Navigate to your Android Studio projects folder
    * Clone the git project `git clone https://github.com/gmsevekari/gajanan_maharaj_sevekari.git`
    * Open the project `open -a "Android Studio" <Path_To_Android Studio projects folder>/gajanan_maharaj_sevekari`
    * Navigate to the project directory and run `flutter pub get`
  * Android Studio
    * Open Android Studio
    * If you are on the Welcome Screen, click the button that says Get from VCS (Version Control System).
      * If you already have a project open: Go to File > New > Project from Version Control...
    * In the "Get from Version Control" window:
      * URL: https://github.com/gmsevekari/gajanan_maharaj_sevekari.git
      * Directory: Click the folder icon ðŸ“ on the right. Navigate to your default Android Studio directory.
      * Important: Android Studio will automatically create a new folder inside that directory with your project name. You don't need to create the new folder manually.
    * Click Clone
    * Android Studio will ask if you want to open the project. Click Trust Project (if asked) and then Open.
    * Click Open (or File > Open).
    * Navigate to the AndroidStudioProjects folder, select the new folder you just cloned, and click OK.
    * Open the pubspec.yaml file in Android Studio.
    * Look for a bar at the top of the editor window that says "Pub get" (or click the download icon).
